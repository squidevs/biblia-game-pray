# Política de Privacidade e LGPD

## Coleta e Uso de Dados

O Bíblia GamePray coleta dados pessoais dos usuários para garantir uma experiência segura, personalizada e funcional, conforme previsto pela Lei Geral de Proteção de Dados (LGPD).

### Quais dados coletamos?
- Nome de usuário e e-mail (identificação e login)
- Dados de perfil (avatar, conquistas, progresso)
- Estatísticas de jogo (nível, moedas, frutos do Espírito, pecados)
- Informações de participação em missões em conjunto
- Dados de ranking e liga (pontuação, posição na leaderboard)

### Por que coletamos esses dados?
- Para autenticar e identificar o usuário
- Para salvar e restaurar o progresso do jogador
- Para permitir participação em missões colaborativas e interações sociais
- Para exibir rankings, ligas e leaderboard de desempenho
- Para garantir a segurança e integridade do jogo
- Para melhorar a experiência do usuário e personalizar conteúdos

### Como usamos os dados?
- Os dados são utilizados apenas para funcionalidades do jogo e nunca vendidos ou compartilhados com terceiros para fins comerciais.
- Informações de ranking e missões colaborativas são exibidas publicamente apenas com o nome de usuário ou apelido escolhido.
- Dados sensíveis (e-mail, senha) são protegidos por criptografia e nunca expostos publicamente.
- O usuário pode editar ou excluir seu perfil a qualquer momento.

### Compartilhamento de dados
- Dados de desempenho (ranking, conquistas) podem ser exibidos em leaderboards e ligas, visíveis para outros jogadores.
- Dados pessoais nunca são compartilhados fora do ambiente do jogo.

### Direitos do usuário
- O usuário pode solicitar acesso, correção ou exclusão de seus dados.
- O usuário pode optar por não participar de rankings públicos ou missões colaborativas.
- Todas solicitações podem ser feitas pelo suporte do jogo.

### Segurança
- Utilizamos criptografia, autenticação segura e boas práticas de proteção de dados.
- Monitoramos e atualizamos constantemente nossos sistemas para garantir a privacidade.

### Atualizações
- Esta política pode ser atualizada conforme novas funcionalidades ou exigências legais. O usuário será avisado sobre mudanças relevantes.
